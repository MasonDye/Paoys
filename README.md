# Paoys

This project is a fork and adaptation based on the projects below.

Paoys is a Windows desktop companion that follows your mouse, sleeps on the taskbar, or roams along it.

### Run

1. Install dependencies: `npm install`
2. Start the app: `npm run start`

### Controls

- Double-click the cat to toggle Sleep mode (sleeps on the taskbar).
- Right-click the cat to toggle kuroneko (invert).
- Use the tray icon to switch variants, choose a mode (Follow Mouse, Sleep, Taskbar Roam), and enable Start at login.
